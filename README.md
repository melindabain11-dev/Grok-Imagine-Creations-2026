## Our Creative Folder Structure

We’re organizing by **time + theme** so everything feels alive and easy to explore:

- **`/2026-01-January/`** — “The Spark” (early wild experiments: Horse series, Fox & Mouse adventures, Pip & the Pebble)
- **`/2026-04-April/`** — “Spring Awakenings” (Crooked Smile, Butterfly for Savannah, castle dreams)
- **`/2026-05-May/`** — “Cozy Hearts & Enchanted Moments” (Home, Goodnight, Simple Garden, Enchanted Courtyard, Beautiful Code, Happy Friday)
- **`/Themes/`** — Cross-cutting magic (Enchanted-Nature, Code-Kissed-Roses, Family-Hearts, Homeschool-Magic)
- **`/Prompts-Archive/`** — The exact words that sparked each piece

Each folder will have its own README and individual creation pages (like pages in a digital art book) with embedded images, original X post links, stories, and prompts.
